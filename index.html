Web Application Development

Course Code: ITM 1131

Course Level: Year One, Semester One

Credit Unit: 4

This course introduces students to Web technologies - HTML, CSS, JavaScript; CSS frameworks; JavaScript frameworks. Students will learn to implement modern web pages using HTML and CSS framework. The course also introduces students to full stack web application programming using JavaScript.

Course Objectives

  1. Describe web technologies (HTML, CSS & JavaScript), CSS frameworks, and JavaScript frameworks.
  2. Implement a simple but modern web page with HTML and CSS.
  3. Build a simple but functional web application using JavaScript.
