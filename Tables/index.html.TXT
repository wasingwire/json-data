<body><h2>Course Code: ITM 1131</h2>
<h2>Course Level: Year One, Semester One</h2>
<h2>Credit Unit: 4</h2><h3>Course Description</h3><p>This course introduces students to Web technologies - HTML, CSS, JavaScript; CSS
frameworks; JavaScript frameworks. Students will learn to implement modern web pages
using HTML and CSS framework. The course also introduces students to full stack web
application programming using JavaScript.</p><h3>Course Objectives</h3><ol>
<li>Describe web technologies (HTML, CSS & JavaScript), CSS frameworks, and
JavaScript frameworks.</li>
<li>Implement a simple but modern web page with HTML and CSS.</li>
<li>Build a simple but functional web application using JavaScript.</li>
</ol><h3>Learning Outcomes</h3><ul>
<li>Use web technologies (HTML, CSS & JavaScript), CSS frameworks, and JavaScript
frameworks.</li>
<li>Design and create a simple but functional web page using HTML and CSS
framework.</li>
<li>Compose and build a simple full stack web application using JavaScript
framework.</li>
